# `@basmilius/validation`

Form validation for personal projects, built on Regle.

## ⭐️ Prerequisites

- Bun >= 1.3.0
- Node >= 25

## 🚀 Getting started

1. Run `bun install` in the root of the project.
2. Run `bun --cwd packages/validation build` to build the project.

## 📦 Notes

- This is the only package that type-checks with `vue-tsc` instead of `tsc`, because it ships Vue single file components.
