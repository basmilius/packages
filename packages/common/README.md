# `@basmilius/common`

Common code shared in personal projects.

## ⭐️ Prerequisites

- Bun >= 1.3.0
- Node >= 25

## 🚀 Getting started

1. Run `bun install` in the root of the project.
2. Run `bun --cwd packages/common build` to build the project.
