# `@basmilius/tools`

Contains commonly used primitives used for building js / ts libraries.

### ⭐️ Prerequisites

- Bun >= 1.2.11
- Node >= 23

### 🚀 Getting started

1. Run `bun install` in the root of the project.
2. Run `bun --cwd packages/tools build` to build the project.
