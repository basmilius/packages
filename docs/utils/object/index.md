# Object

Helpers for inspecting prototype chains and assigning methods or properties on objects with explicit semantics.

- [`getPrototypeChain`](/utils/object/getPrototypeChain) — collects accessor descriptors from a class' prototype chain.
- [`setObjectMethod`](/utils/object/setObjectMethod) — installs a function on a class' prototype.
- [`setObjectValue`](/utils/object/setObjectValue) — defines a non-enumerable, non-writable property on an object.
