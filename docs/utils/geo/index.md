# Geo

Two small geographic helpers built on top of an internal country lookup table.

- [`isNorthernHemisphere`](/utils/geo/isNorthernHemisphere) — checks whether a country sits in the northern hemisphere.
- [`isPointInPolygon`](/utils/geo/isPointInPolygon) — ray casting point-in-polygon test for arbitrary 2D polygons.
